# vue-cli

A simple CLI for scaffolding node test suite.

### Installation

Prerequisites: [Node.js](https://nodejs.org/en/) (>=4.x) .

``` bash
$ npm install -g souche-test-cli
```

### Usage

#### Init
``` bash
$ stest init
```

Example:

``` bash
$ stest init
```

#### New case
``` bash
# 在当前目录创建测试文件
$ stest newcase <test-case-name>
```

Example:

``` bash
$ stest newcase index
```
