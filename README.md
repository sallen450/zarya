# zarya

A simple CLI for scaffolding node test suite.

### Installation

Prerequisites: [Node.js](https://nodejs.org/en/) (>=4.x) .

``` bash
$ npm install -g zarya
```

### Usage

#### Init
``` bash
$ zarya init
```

Example:

``` bash
$ zarya init
```

#### New case
``` bash
# 在当前目录创建测试文件
$ zarya newcase <test-case-name>
```

Example:

``` bash
$ zarya newcase index
```
